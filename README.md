# Controle de Estoque em Python

Sistema simples de controle de estoque desenvolvido em Python com o objetivo de praticar lógica de programação e manipulação de dados.

# Funcionalidades

- Cadastro de produtos
- Consulta de produtos no estoque
- Atualização de quantidade
- Organização de informações de estoque

# Tecnologias utilizadas
- Python

# Como executar o projeto

1. Clone o repositório
2. Abra a pasta no VS Code
3. Execute o arquivo:

```bash
python controle_estoque.py
